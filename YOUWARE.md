# Spirit Tech Revolution (STR) — Vitrine Professionnelle

## Description
Vitrine professionnelle pour Spirit Tech Revolution (STR), une entreprise de services numériques, avec une palette de verts doux et un design moderne.

## Project Status
- **Type**: React + TypeScript + Tailwind CSS
- **Build**: Vite 7.0
- **Styling**: Tailwind CSS avec palette personnalisée (mint/sage)

## Pages / Sections
- **Hero** — Section d'accueil avec statistiques et CTA
- **Services** — 4 services génériques : Solutions Sur Mesure, Expériences Numériques, Intégration & Infrastructure, Innovation & Transformation
- **À Propos** — Pourquoi nous choisir (4 avantages)
- **Contact** — Formulaire avec email professionnel + coordonnées
- **Footer** — Liens, mentions légales, retour en haut

## Design System
- **Couleurs**: Palette `mint` (verts doux) et `sage` (verts/gris naturels)
- **Typographie**: DM Serif Display (titres) + Plus Jakarta Sans (corps)
- **Composants**: Glass-morphism cards, gradient accents, animations au scroll
- **Responsive**: Mobile-first, breakpoints sm/md/lg/xl

## Fichiers Clés
- `src/components/Header.tsx` — Navigation sticky avec logo STR
- `src/components/Hero.tsx` — Section accueil avec animations
- `src/components/Services.tsx` — Grille des 4 services
- `src/components/About.tsx` — Section pourquoi nous
- `src/components/Contact.tsx` — Formulaire + infos contact
- `src/components/Footer.tsx` — Pied de page avec logo STR
- `tailwind.config.js` — Configuration couleurs mint/sage

## Tech Stack (utilisé en interne, non mentionné sur le site)
- **Backend**: React, Vite, TypeScript, Tailwind CSS
